JEP 472는 Java Development Kit (JDK)의 12번째 모듈로, JDK 24에서 JNI(Java Native Interface) 사용을 제한하기 위한 조치를 취할 준비를 합니다. JNI는 Java와 네이티브 코드 간의 인터페이스를 제공하며, 이를 통해 Java 애플리케이션이 네이티브 라이브러리에 접근할 수 있습니다. 그러나 JNI는 보안 문제와 호환성 문제 등의 이유로, 미래의 Java 플랫폼에서 사용되는 것을 권장하지 않고 있습니다.

JEP 472는 JDK 24에서 JNI 사용을 제한하기 위해 어떤 조치를 취할지 결정할 것입니다. 이를 통해 Java 개발자들은 JNI를 사용하는 코드의 안전성과 유지보수성을 고려하여 적절한 대체 방법을 선택할 수 있습니다.

JEP 472는 Java SE Platform Specification에서 설명되며, JDK의 jdk-12 브랜치에서 진행중입니다. 이 모듈은 OpenJDK 프로젝트에서 관리되고 있으며, Java SE Platform Specification에서 자세한 내용을 확인할 수 있습니다.
