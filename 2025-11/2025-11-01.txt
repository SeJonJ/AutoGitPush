```
string text = "Line 1 \n Line 2";
text = text.Replace("\n", " ");
```

This will result in `"Line 1 Line 2"`.

If you want to replace both line breaks and carriage returns with a space, you can use the following code:

```
string text = "Line 1 \r\n Line 2";
text = text.Replace("\n", " ").Replace("\r", " ");
```

This will result in `"Line 1 Line 2"` as well.

You can also use the `String.Join()` method to join all lines together with a space:

```
string text = String.Join(" ", new string[] { "Line 1", "Line 2" });
```
